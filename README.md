#Ejecutar en desarrollo
1. Clonar el repositorio con el comando: "git clone https://......."

2. Ejecutar el comando:
````
yarn install
````
3. Intalar nest cli: 
````
npm i @nestjs/cli
````
4. Descargar la imagen mongo de Docker

5. Levantar la base de datos con docker:
````
docker-compose up -d
````
